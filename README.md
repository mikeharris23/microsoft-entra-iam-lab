# Microsoft Entra IAM Lab

This project demonstrates the design and implementation of an enterprise
identity and access management environment using Microsoft Entra ID.

The lab models a fictional financial-services company and includes identity
lifecycle management, role-based access control, multifactor authentication,
Conditional Access, privileged identity management, access reviews, and
automation.

## Completed Modules

### Module 1 — Enterprise Identity Foundation

- Created cloud-only workforce identities
- Configured user attributes and manager relationships
- Created department and business-role security groups
- Separated standard and privileged administrator accounts
- Applied least-privilege role assignment
- Created a scoped administrative unit
- Developed an identity inventory and access matrix
