# Naming Conventions

## User Principal Names

Standard users:
firstname.lastname@tenantdomain

Administrative users:
adm-firstname.lastname@tenantdomain

Service accounts:
svc-purpose@tenantdomain

Emergency access accounts:
breakglass01@tenantdomain

## Security Groups

SG-DEPT-[Department]-Users
SG-ROLE-[BusinessRole]
SG-APP-[Application]-[AccessLevel]
SG-ADMIN-[AdministrativeRole]

## Administrative Units

AU-[DepartmentOrRegion]

## Conditional Access Policies

CA-[Sequence]-[Purpose]-[Target]

## Access Packages

AP-[Department]-[Purpose]
